About MyWeather

🌤️ What is MyWeather?

MyWeather is a sleek, modern weather web app that gives you real-time weather updates with a simple and intuitive interface. Whether you're planning a trip, checking your hometown's forecast, or just curious about current conditions—MyWeather has you covered.

🛠️ Features

✅ Real-time weather data using OpenWeatherMap API
✅ Search by city or use your current location
✅ Smart weather tips based on temperature
✅ Weather highlights of major Indian cities
✅ Clean UI with responsive design

👨‍💻 About the Developer

My name is Krish Modh, and I built MyWeather as a project to enhance user experience through weather tracking. It combines JavaScript, HTML, and CSS with external APIs to deliver an engaging, informative, and responsive app. Always improving—more features like forecast and maps coming soon!
