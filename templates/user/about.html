About Weather App

Your reliable companion for accurate weather updates worldwide.

🌎 Our Mission

Weather App was built to provide users with real-time weather data, detailed forecasts, and interactive tools—all wrapped in a modern, user-friendly interface. Whether you're planning a trip, checking local conditions, or tracking severe weather, we’ve got you covered.

⚡ Features

  • Live weather updates using OpenWeather API
  • 7-day and hourly forecasts with interactive visuals
  • Light/Dark mode for a personalized experience
  • Save favorite locations for quick access
  • Responsive design that works on all devices

💡 Technology Stack

Built using Flask for the backend, Oracle Database for secure storage, and modern HTML, CSS, and JavaScript for a smooth and responsive frontend experience.
